# Contributing

Thanks for your interest in contributing to second-opinion-mcp-server!

## Development Setup

1. Fork and clone the repo
2. Install dependencies: `npm install`
3. Build: `npm run build`
4. Run tests: `npm test`

Requires Node.js 24+. Use `nvm install` to pick up the version from `.nvmrc`.

## Making Changes

1. Create a branch from `main`
2. Make your changes
3. Add or update tests as needed
4. Ensure `npm run build` and `npm test` pass
5. Open a pull request

## Reporting Issues

Use [GitHub Issues](https://github.com/sofus-nl/second_opinion/issues) to report bugs or request features. Include:

- Steps to reproduce (for bugs)
- Expected vs actual behavior
- Node.js version and OS

## Code Style

- TypeScript strict mode
- ESM modules (`.js` extensions in imports)
- Keep it simple
